微软拼音配置小鹤双拼
2020-04-08 22:56 folder 工具   label 输入法   501 words

操作环境：Win10 版本1903

背景

之前进入git bash，谷歌拼音输入法无法进行中英状态切换，而微软拼音可以正常进行中英切换，所以舍弃谷歌拼音了。我长期以来使用的是小鹤双拼，所以需要对微软拼音进行配置小鹤双拼。

设置的路径

设置–时间和语言–语言–首选语言–选项–微软拼音–常规–添加双拼方案

然后手动逐个输入双拼的键位方案。

另一种不需要手动逐个输入的方法是修改注册表。

Win+R -> regedit

进入注册表，定位到如下位置

计算机\HKEY_CURRENT_USER\Software\Microsoft\InputMethod\Settings\CHS

新建一个字符串

名称：UserDefinedDoublePinyinScheme0
数值：小鹤双拼2^*iuvdjhcwfg^xmlnpbksqszxkrltvyovt

新建完成后，再次打开上面的设置路径，即可设置小鹤双拼。

生效注册表

如果发现未生效的话，可以进行如下任一操作（重启资源管理器）：

  • 命令行修改

Win+R -> cmd
taskkill /IM explorer.exe

  • 任务管理器修改

Ctrl+Alt+Del -> 任务管理器 -> 详细 -> explorer.exe -> 右键结束任务 -> 文件 -> 运行新任务 -> explorer.exe -> 确定
